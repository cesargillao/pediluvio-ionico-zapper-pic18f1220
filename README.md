# Pediluvio Iónico con Zapper (PIC18F1220)

## Se proporciona:
* Diagrama esquemático para emular el funcionamiento (Proteus)
* Diagrama esquemático y PCB (CadSoft Eagle)
* Software en C y ASM con su respectivo HEX (CCS Compiler)
